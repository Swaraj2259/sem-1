= 📂 Repository Structure  

- python

- C++

- Assignments

 💻 Prerequisites  

Ensure you have the following tools installed to run the code:  
- Python(v3.8 or higher)  
- C++ Compiler** (e.g., GCC)  



 🛠 How to Run  

 Python Code  
1. Navigate to the `Python` directory.  
2. Run any `.py` file using the command:  
   ```bash
   python filename.py

C++ Code

	2.	Compile the file using:

g++ filename.cpp -o output


	3.	Run the compiled file:

./output
